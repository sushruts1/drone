# Autonomous Drone Development

This project focuses on designing and building a quadcopter drone using individual hardware components. 
The goal of the project is to understand drone flight systems and eventually develop autonomous flight capabilities.

## Components

- Brushless DC motors
- Electronic Speed Controllers (ESC)
- Flight controller
- LiPo battery
- Drone frame
- Propellers

## Objectives

- Assemble and configure a stable quadcopter drone
- Understand flight control and power distribution systems
- Integrate sensors and microcontrollers
- Explore autonomous navigation capabilities

## Current Status

The project is currently in progress and focuses on hardware assembly and system integration.

## Future Plans

- Implement autonomous navigation
- Integrate sensors for obstacle detection
- Improve flight stability and control algorithms
